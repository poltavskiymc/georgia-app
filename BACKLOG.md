## Согласовано сделать (приоритет сверху)

1. **Погода по городам** — прогноз для Тбилиси/Батуми и др. через **open-meteo.com** (бесплатно, без ключа). Полезно на дорогу и open-air концерт. Логичное место — блок на «Гид → Места» или карточка на «Деньги» (там уже «Практика»). Не забыть `addTraffic()`.

2. **Вкладка «Деньги»: мультивалютный конвертер + инфо-плашка в «?»** (согласовано 2026-07-12)
   - **Инфо-плашка** `<p class="warn">` («Онлайн — тянем реальный курс…», в карточке «💸 Конвертер»
     в `index.html`) занимает место — убрать из потока, вынести в значок **«?»** рядом с заголовком
     карточки; по нажатию разворачивается (проще всего `<details>`, как в TIPS, или toggle класса).
   - **Валюты: GEL + RUB + USD + EUR** вместо нынешней пары лари⇄рубль (`js/money.js`).
     Курс тянется одним запросом: `https://open.er-api.com/v6/latest/GEL` уже отдаёт `rates.RUB/USD/EUR`
     в том же ответе — доп. запросов и трафика не нужно, `addTraffic()` остаётся как есть.
   - **Ручной ввод курса — для каждой валюты** (обменник на границе даёт свой), приоритет ручного над
     автоматом сохраняется. Хранение в localStorage: сейчас плоские ключи `gel_rate*` — переделать в
     объект вида `{RUB:{val,src,ts}, USD:{…}, EUR:{…}}` (курсы держать как «1 GEL = X»), с миграцией со
     старых ключей при первом запуске.
   - **Пересчёт во все стороны сразу:** вбил рубли — тут же пересчитались лари, доллары и евро; вбил евро —
     пересчитались остальные три. Т.е. поле, в которое печатают, = источник; все прочие перерисовываются
     (через базовую валюту GEL). Кнопка ⇅ (`#swap`) при 4 полях теряет смысл — убрать.

## Грузинская озвучка (отложено 2026-07-12, решили пока жить с транскрипцией)

Сейчас: грузинского голоса нет ни в iOS (Apple его не поставляет), ни в Android-наборе по умолчанию —
поэтому `speak()` в `js/util.js` при отсутствии ka-голоса читает **русскую транскрипцию русским голосом**.

**Проверено и не работает:** бесплатный хак `translate.google.com/translate_tts` грузинский НЕ отдаёт
(`tl=en` → 200 `audio/mpeg`, `tl=ka` → **400**). У Google Translate для грузинского нет TTS в принципе.

Варианты, если вернёмся (в порядке предпочтения):
1. **47 mp3 в репо** — сгенерить один раз нейродвижком (Meta MMS `facebook/mms-tts-kat`, бесплатно, без
   регистраций), закоммитить, добавить в `ASSETS` в `sw.js`. Вес: ~5–8 КБ на фразу → **300–600 КБ на всё**.
   Офлайн, без ключей, без трафика. Минус: только фиксированные фразы, переводчик так не озвучить.
2. **Azure Speech по ключу** (`ka-GE-EkaNeural`/`ka-GE-GiorgiNeural`) — нейроголос для любого текста,
   включая переводчик; free tier 500k символов/мес, но регистрация с картой + нужен интернет.
3. **espeak-ng в WASM** — ~400 КБ, офлайн, любой текст, но голос откровенно роботизированный.

## Процесс (важно для следующего чата)

- Правишь css/js → подними номер в ТРЁХ местах, они должны совпадать: `VER` в `sw.js`, `?v=N` у всех `<link>/<script>` в `index.html`, `APP_VERSION` в `js/settings.js`. Новый файл — ещё и в список `VERSIONED` в `sw.js`. HTML грузится network-first, css/js — SWR с `?v=N`: залипнуть на старой версии больше нельзя (сборка 21 залипла именно так).
- Проверка без браузера: `node --check js/*.js` + прогон настоящего `index.html` в **jsdom** (`npm i jsdom` в скретчпаде, не в репо; скрипты подключать реальными `<script>`, не через `eval` — иначе top-level `const` из `data.js` не виден). **Headless Chrome тут зависает — не использовать.**
- Деплой: `git add -A && git commit && git -c credential.helper='!gh auth git-credential' push origin main`. Pages пересобирается ~1 мин. Проверка live: `curl .../js/settings.js | grep сборка`.
- Архитектура и конвенции — в `README.md`.
